**일반적인 REST API에서 게시글(Post)에 대한 각각의 CRUD에 대응되는 HTTP  methods와 Url을 작성하시오**.

1) **리소스의 목록**

* HTTP methods: `GET`
* URL : `api/v1/student`

2) **리소스 생성**

- HTTP methods: `POST`
- URL : `api/v1/create`

3) **리소스 중 하나 표시**

- HTTP methods: `GET`
- URL : `api/v1/student/1`

4) **리소스 수정**

- HTTP methods: `PUT`
- URL: `api/v1/student/{student_id}`

5) **리소스의 삭제**

- HTTP methods: `DELETE`
- URL : `api/v1/student/{student_id}`
